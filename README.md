# SistemadeGerenciamentodeEventosAcademicos
Projeto contendo manipulações de informações e CRUDs envolvendo banco de dados

Este projeto consiste na criação e utilização de um banco de dados para funcionar como um protótipo de um sistema de gerenciamento de eventos acadêmicos, onde usuários podem se inscrever em eventos, verificar seus certificados adquiridos, e administradores podem criar novos eventos, emitir certificados, entre outras ações. (Todas as informações inseridas são de uso meramente ilustrativo, não representando nenhum evento, pessoa ou situação real)
